# DAproject1

## 📊 О проекте
Данный проект посвящён анализу данных о продажах.  
Цель работы — исследовать структуру продаж, поведение клиентов и выявить ключевые бизнес-закономерности.

---

## 🧹 Этапы работы

### 1. Предобработка данных
- анализ пропусков
- обработка отсутствующих значений
- проверка и исправление некорректных данных
- анализ выбросов

### 2. Исследовательский анализ данных (EDA)
Проведён анализ:
- продаж по категориям товаров
- активности клиентов по регионам
- поведения клиентов по возрасту и полу
- сезонности продаж
- повторных покупок

### 3. Визуализация данных
Использованы графики:
- barplot
- lineplot
- scatterplot

---

## 📈 Основные выводы
- выраженной сезонности продаж не выявлено  
- наибольшая выручка формируется отдельными категориями товаров  
- поведение клиентов различается в зависимости от возраста и региона  
- скидки не показывают сильной связи с ростом продаж  
- часть товаров формирует основную долю повторных покупок  

---

## 🛠 Используемые технологии
- Python  
- Pandas

- Seaborn  

---

## 📌 Итог
Проведённый анализ позволил выявить ключевые закономерности продаж и поведение клиентов, которые могут быть использованы для оптимизации бизнес-стратегии и маркетинга.
